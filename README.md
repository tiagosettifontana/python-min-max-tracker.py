# Python Min-Max Tracker 📊

This is a Python program that repeatedly prompts the user for integer numbers and, upon completion, calculates the **maximum** and **minimum** numbers entered.

This project was developed as part of Exercise 5.2 from the "Python for Everybody" (PY4E) course by the University of Michigan.

## 🎯 Features

* **Indefinite Loop:** The program runs continuously, requesting data until the exit keyword (`done`) is entered.
* **Error Handling:** Uses `try/except` blocks to catch and gracefully handle invalid inputs (e.g., if the user types text instead of numbers).
* **Data Analysis:** Initializes variables with `None` and uses conditional logic (`if/elif`) to track the maximum and minimum values in real-time.

## 🛠️ Technologies & Concepts Applied

* Language: **Python 3**
* Concepts:
  * Loops (`while True`)
  * Control Flow (`break`, `continue`)
  * Exception Handling (`try`, `except`)
  * Data Types (`NoneType`, `int`, `str`)
  * Identity Operators (`is`)
